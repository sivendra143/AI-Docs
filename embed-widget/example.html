Get Started

Add the PDF Chat Widget to your website in just a few simple steps:

1. Include the widget script

Add this script tag to your HTML, just before the closing </body> tag:

html 
<script src="https://your-domain.com/path/to/pdf-chat-widget.min.js"></script>

2. Initialize the widget

Add this script after including the widget, customizing the options as needed:

javascript 
<script>
    document.addEventListener('DOMContentLoaded', function() {
        new PDFChatWidget({
            apiUrl: 'http://your-api-url.com',  // Your PDF Chat API URL
            position: 'bottom-right',           // Widget position
            primaryColor: '#4f46e5',           // Your brand color
            title: 'Chat with Us'               // Widget title
        });
    });
</script>
Tip: Make sure to replace http://your-api-url.com with your actual PDF Chat API URL.

Configuration Options

Customize the widget's appearance and behavior with these options:

javascript 
{
    apiUrl: 'http://your-api-url.com',  // Required: Your PDF Chat API URL
    position: 'bottom-right',           // Widget position: 'bottom-right', 'bottom-left', 'top-right', or 'top-left'
    primaryColor: '#4f46e5',           // Primary color for the widget
    title: 'Chat with Us',              // Title displayed in the widget header
    welcomeMessage: 'Hello! How can I help you today?',  // Optional welcome message
    avatar: 'https://example.com/avatar.png',  // Optional URL to an avatar image
    zIndex: 9999                       // CSS z-index of the widget
}
