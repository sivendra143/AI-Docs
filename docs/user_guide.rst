Overview
--------

AI Document Chat allows you to upload documents and interact with them using natural language. The application uses advanced language models to understand your questions and provide relevant answers based on the content of your documents.

Key Features
-----------

- **Document Upload**: Upload multiple document formats (PDF, DOCX, TXT, etc.)
- **Natural Language Processing**: Ask questions in plain English
- **Conversation History**: Maintain context across multiple questions
- **Source References**: See which parts of your documents were used to generate answers
- **Web Interface**: Easy-to-use web interface
- **API Access**: Programmatic access to all features

Quick Start
-----------

1. **Upload Documents**
   - Click the "Upload" button in the sidebar
   - Select one or more documents to upload
   - Wait for the documents to be processed

2. **Start Chatting**
   - Type your question in the chat input
   - Press Enter or click the send button
   - The AI will process your question and provide an answer

3. **Manage Conversations**
   - Start a new conversation at any time
   - View and switch between previous conversations
   - Delete conversations when no longer needed
